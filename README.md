<h3>About</h3>
A lightweight CSV Viewer using C++ and QT

<h3>Build</h3>
```bash
qmake -r CSVView.pro -config debug
make
```

<h3>Run</h3>
```bash
./csvview <filename>
````
Or double click on the icon
